- Mobile Navigation
- SEO-Friendly Structure
- Fast Loading

## Deployment

### GitHub Pages

1. Create a GitHub repository
2. Upload all files
3. Go to:

Settings → Pages

4. Source:
   Deploy from branch

5. Branch:
   main

6. Save

Your website will be live at:

https://yourusername.github.io/repository-name
